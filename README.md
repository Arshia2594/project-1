# Project Frontend

This project is initialized using **React + Vite** and styled using **Tailwind CSS**.

---

##  Tech Stack
- React JS
- Vite
- Tailwind CSS

---

## Getting Started

###  Clone the Repository
git clone https://github.com/Arshia2594/Project-Frontend.git

###  Install Dependencies
npm install

###  Run the Project
npm run dev
